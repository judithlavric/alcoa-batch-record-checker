# alcoa-batch-record-checker

**Automated ALCOA+ compliance checker for GMP executed batch records in clinical biologics manufacturing.**

---

## Context

In GMP-regulated biologics manufacturing, every entry in an executed batch record must satisfy the **ALCOA+ principles**: Attributable, Legible, Contemporaneous, Original, Accurate, Complete, Consistent, Enduring, and Available. These principles are mandated by EU GMP Chapter 4, ICH Q7, FDA 21 CFR Parts 11 and 211, and WHO Technical Report Series guidance.

This tool simulates the GDP documentation review workflow used by QA/Documentation Specialists during batch disposition review — a core responsibility in clinical-phase manufacturing environments such as upstream biologics processing (USP), downstream processing (DSP), and buffer/media preparation.

---

## What this tool does

1. **Reads** an executed batch record CSV (`batch_record_USP_001.csv`) representing a 200L PBS 1X buffer preparation for a Phase II clinical biologic
2. **Evaluates** every step against 9 ALCOA+ principles using `checker.py`
3. **Generates** a professional, self-contained HTML review report via `report_generator.py`

Each finding includes:
- Step ID and ALCOA+ principle violated
- Severity classification (CRITICAL / MAJOR / MINOR)
- Observation text written in GDP finding style
- Specific regulatory reference (EU GMP / ICH Q7 / 21 CFR)

## How to run

**Requirements:** Python 3.10+, pandas

```bash
# Install dependency
pip install pandas

# Step 1: Run the ALCOA+ checker
python checker.py

# Step 2: Generate the HTML report
python report_generator.py

# Step 3: Open the report
open output/ALCOA_Review_Report_USP001.html
```

## Intentional ALCOA+ violations in the batch record

| Step | Principle | Severity | Violation |
|------|-----------|----------|-----------|
| USP-006 | **Complete** | MAJOR | `checked_by` field is blank — second-person verification missing |
| USP-010 | **Attributable** | CRITICAL | Operator `OP-999` is not in the authorised operator register |
| USP-010 | **Accurate** | CRITICAL | Pre-adjustment pH of 7.21 is outside spec 7.35–7.45 |
| USP-012 | **Accurate** | CRITICAL | Post-adjustment pH of 7.52 is outside spec 7.35–7.45 |
| USP-017 | **Complete** | MAJOR | `checked_by` field blank on bioburden sampling step |
| USP-017 | **Legible** | MAJOR | `actual_value` = `PENDING` — result not recorded at time of execution |
| USP-018 | **Available** | MAJOR | `bom_reference` field is blank — required BOM reference missing |
| USP-020 | **Accurate** | CRITICAL | Post-use filter bubble point 33.1 psi < minimum 35 psi — sterility assurance failure |
| USP-021 | **Consistent** | CRITICAL | Filtered volume 174.2 L vs pre-filtration 199.8 L — 12.8% discrepancy exceeds 5% tolerance |
| USP-023 | **Contemporaneous** | CRITICAL | Step recorded on 2024-03-17, outside production window (2024-03-15) — indicates late entry |
